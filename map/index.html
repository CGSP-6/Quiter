How we developed these maps

Project Architecture

Our 4 developers collaborate to develop programs through github, and use weekly meetings through zoom. The framework of the webpage was drafted by Shiqi, and Nabilah, Namrata, and Pelinsu modified the webpage. The map was developed by Shiqi.

The map program is divided into three parts: front end: web page, back end: Python, Database: Oracle.

  • The first part, HTML, Javascript, CSS and related resource files constitute the basic framework of the web page.
  • In the second part, write a python program in the cgi-bin folder, use folium to generate a map, and use cx_Oracle to link to the Oracle database.
  • In the third part, use Oracle to write a data table named CGSP_Point, the fields are ID, TYPE, LON_X, LAT_Y, where ID is the primary key. And by creating a sequence, the automatic growth of the primary key ID is realized.
